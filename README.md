## ğŸ¨ Customization

### Theme Colors

Customize colors by modifying the CSS variables in `app/globals.css`:

```css
:root {
  --background: 0 0% 100%;
  --foreground: 240 10% 3.9%;
  --primary: 240 5.9% 10%;
  /* Add more custom colors */
}
```
